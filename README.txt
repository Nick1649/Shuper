---------------
Shuper - Super Shopper Self-Checkout
---------------

Shuper is an innovative self-checkout mechanism that allows all smartphone users to be able to accelerate payments at retail stores.
This product allows merchants to use similar self-checkout mechanism that large grocery stores implement.


Shuper.ios

The main implementation of the Shuper application.

Shuper.web

The backend of the applications. 
It contains the database as well as the location of the Simplify and MasterPass APIs.

Shuper.android

An early version of the Shuper application.
